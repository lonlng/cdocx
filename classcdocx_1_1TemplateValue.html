详细描述

A polymorphic value for template population

Represents either text or image content, with optional formatting and sizing. Created via static factory methods and configured through chainable setters.

Usage:
auto text = cdocx::TemplateValue::text("Hello World")
.with_format(cdocx::TemplateFormat().bold().size(24));
auto img = cdocx::TemplateValue::image("chart.png")
.sized(400, 300)
.centered()
.with_caption("Figure 1");
