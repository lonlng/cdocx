详细描述

High-level API for bookmark-based content replacement

Provides a convenient interface for replacing bookmark content with text or images while preserving or controlling formatting.

Features:
  • Text replacement with format preservation
  • Batch text replacement
  • Image insertion with automatic caption generation
  • Alignment control for images
  • Conditional replacement via callbacks
  • Detailed statistics tracking
