版本
0.8.0

CDocx is a modern C++17 library for creating, reading, and writing Microsoft Office Word (.docx) files.

Usage: #include <cdocx.h> using namespace cdocx;

// Create a new document auto doc = CDocx::create_document();

// Add content using fluent API doc->add_paragraph("Hello, World!") ->add_run(" Bold text", Font().set_bold(true));

// Save the document doc->save("output.docx");
