详细描述

Programmatically insert bookmarks around existing text in a document.

Searches for the first occurrence of a text string in the document and wraps it with <w:bookmarkStart> / <w:bookmarkEnd> markers. Handles text that spans multiple adjacent runs within a paragraph.

Usage Example:
cdocx::Document doc("report.docx");
doc.open();
cdocx::BookmarkInserter inserter(&doc);
inserter.insert("COMPANY", "Acme Inc.");
inserter.insert("DATE", "2024-01-15");
doc.save("template.docx");
