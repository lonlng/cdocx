Template engine for placeholder replacement in DOCX documents

The Template class provides a powerful mechanism for replacing placeholders in Word documents. It uses a Finite State Machine (FSM) to handle placeholders that may be split across multiple text runs.

Supported Placeholder Formats:
  • Default format: {{key}}
  • Custom format: set prefix and suffix via set_pattern()
Processing Flow:
  1. Set placeholder values (set(), set_image())
  2. Execute replacement (replace_all())
  3. Save document
Notes:
  • Placeholder matching is case-sensitive
  • Supports placeholders across multiple runs
  • Image placeholders are replaced with actual images
参见
Document, Document::get_paragraphs()
