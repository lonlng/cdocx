详细描述

Comprehensive text formatting properties

Inspired by minidocx, provides detailed control over text appearance. All properties are optional - unset properties will not be applied.

Usage Example:
TextProperties props;
props.font = TextProperties::Font{"Arial", "SimHei"};
props.font_size = 24; // 12pt
props.color = "FF0000";
props.highlight = TextProperties::Highlight::Yellow;
run.set_properties(props);
