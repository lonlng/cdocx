详细描述

Helper class for applying text formatting to XML run elements

This class provides static methods to apply various text formatting attributes to a w:r (run) element. It centralizes the formatting logic to ensure consistency between Run and Paragraph classes.

Usage Example:
pugi::xml_node run = ...; // Get a run element
// Apply formatting
TextFormatContext::apply_color(run, "FF0000");
TextFormatContext::apply_font_size(run, 24);
TextFormatContext::apply_bold(run, true);
