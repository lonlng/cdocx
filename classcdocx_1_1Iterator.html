template<class T, class P, class C>
class cdocx::Iterator< T, P, C >

Template iterator for CDocx collections

Template iterator for traversing CDocx document elements

模板参数
TElement type
PParent node type
CCurrent node type
参见
Iterator
模板参数
TThe element type (Run, Paragraph, Table, etc.)
PThe parent node type (typically pugi::xml_node)
CThe current node type (typically pugi::xml_node)

This iterator supports C++11 range-based for loops and provides a standard iterator interface. It wraps pugixml nodes and converts them to CDocx element types on demand.

Iterator Category:
  • Forward Iterator
Requirements:
  • Type T must provide set_parent() and set_current() methods
  • Type T must have a default constructor
