Unified dictionary-style template engine

The one-stop API for template population. Supports both placeholder ({{key}}) and bookmark-based replacement.

Dictionary Style:
cdocx::TemplateEngine engine(&doc);
engine["name"] = cdocx::TemplateValue::text("John");
engine["logo"] = cdocx::TemplateValue::image("logo.png");
engine.apply();
