File format detection and conversion utilities

版本
0.8.0

Provides utility methods for working with file formats, such as detecting file format or converting file extensions to/from file format enums.

Usage Example:
auto info = cdocx::FileFormatUtil::detect_file_format("document.docx");
if (info->load_format() == cdocx::LoadFormat::Docx) {
std::cout << "It's a DOCX file!" << std::endl;
}
