# CDocx v0.3.0 更新总结

**发布日期**: 2026-02-11  
**版本**: v0.3.0  
**更新类型**: 功能增强版本

---

## 概述

CDocx v0.3.0 实现了完整的纯 C++ 书签替换功能，摆脱了对 Aspose.Words 的依赖。本版本基于 `docs/CDOCX_OPTIMIZATION_PLAN.md` 技术方案实现，包含图片插入、格式保留、标题生成等核心功能。

---

## 主要更新内容

### 1. BookmarkReplacer 类（核心功能）

**文件**: `include/cdocx/bookmark_replacer.h`, `src/bookmark_replacer.cpp`

提供了高层 API 用于书签替换操作：

- **文本替换**:
  - `replace_text()` - 简单替换，保留原格式
  - `replace_text_batch()` - 批量替换多个书签
  - `replace_text_with_format()` - 指定格式替换

- **图片替换**:
  - `replace_with_image()` - 简单图片替换（默认居中，可选标题）
  - `replace_with_image_advanced()` - 高级图片替换（支持尺寸和对齐）
  - `replace_with_image_from_memory()` - 从内存插入图片

- **高级功能**:
  - `replace_if()` - 条件替换（使用回调函数）
  - `replace_and_remove()` - 替换后删除书签标记
  - `has_bookmark()` - 检查书签存在
  - `list_bookmarks()` - 列出所有书签
  - `get_stats()` - 获取替换统计信息

### 2. CaptionGenerator 类（图标题生成）

**文件**: `include/cdocx/caption_generator.h`, `src/caption_generator.cpp`

生成符合中文标准的图标题：

- **格式**: "图 X [描述]"
- **字体**: 中文宋体(SimSun)，英文 Times New Roman
- **字号**: 10.5pt（五号）
- **对齐**: 居中

**API**:
- `insert_figure_caption()` - 在指定段落后插入标题
- `insert_figure_caption_at_end()` - 在文档末尾插入标题
- `count_existing_figures()` - 统计文档中已有图片数量
- `get_next_figure_number()` - 获取下一个图片编号
- `is_figure_caption()` - 检测段落是否为图标题

### 3. Bookmark 类增强

**文件**: `include/cdocx/advanced.h`, `src/advanced.cpp`

新增结构和成员函数：

- **BookmarkFormat 结构体**: 存储字符格式信息
  - `font_ascii`, `font_far_east`, `font_hansi` - 字体设置
  - `font_size` - 字号（半磅）
  - `color` - 颜色（RGB）
  - `bold`, `italic`, `underline`, `strikethrough` - 格式标志

- **新增成员函数**:
  - `get_format()` - 提取书签范围内的格式
  - `set_text_keep_format()` - 保留格式设置文本
  - `set_text_formatted()` - 使用指定格式设置文本
  - `is_cross_paragraph()` - 检查是否跨段落
  - `get_covered_paragraphs()` - 获取书签覆盖的所有段落
  - `set_text_cross_paragraph()` - 跨段落设置文本

### 4. 图片对齐支持

**枚举**: `ImageAlignment`
- `Left` - 左对齐（使用 anchor 定位）
- `Center` - 居中对齐（使用 inline 定位）
- `Right` - 右对齐（使用 anchor 定位）

**结构体**: `ImageSize`
- 支持磅(pt)和 EMU 单位转换
- 1 磅 = 12700 EMU
- 自动尺寸验证

## 使用示例

### 基本文本替换

```cpp
#include <cdocx.h>
#include <cdocx/bookmark_replacer.h>

cdocx::Document doc("template.docx");
doc.open();

cdocx::BookmarkReplacer replacer(&doc);

// 简单替换（保留格式）
replacer.replace_text("NAME", "张三");
replacer.replace_text("DATE", "2024-06-15");

// 批量替换
std::map<std::string, std::string> data = {
    {"COMPANY", "北京地质研究院"},
    {"PROJECT", "地铁隧道检测"}
};
replacer.replace_text_batch(data);

doc.save("output.docx");
```

### 图片替换与标题

```cpp
// 简单图片替换（自动居中，带标题）
replacer.replace_with_image("RESULT_IMG", "gpr_result.png", 
                            "地质雷达检测结果");

// 高级图片替换（指定尺寸和对齐）
cdocx::ImageSize size(450, 300);  // 宽450pt，高300pt
replacer.replace_with_image_advanced("CHART", "analysis_chart.jpg", 
                                      size, "数据分析图",
                                      cdocx::ImageAlignment::Center);
```

### 保留格式替换

```cpp
// 获取原格式并修改
cdocx::BookmarkFormat fmt;
fmt.font_far_east = "宋体";
fmt.font_size = 24;  // 12pt
fmt.bold = true;

replacer.replace_text_with_format("TITLE", "最终报告", fmt);
```

## 已知限制

1. **图片格式支持**: 优先支持 PNG/JPG，其他格式建议先转换
2. **跨段落书签**: 跨多段书签替换时，中间段落会被删除合并
3. **内存图片**: 从内存插入图片需要预先准备关系ID
