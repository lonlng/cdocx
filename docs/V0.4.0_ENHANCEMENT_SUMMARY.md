# CDocx v0.4.0 增强功能总结

## 概述

CDocx v0.4.0 借鉴了 minidocx 的优秀设计，在保持 C++17 兼容的前提下，大幅增强了文本和段落格式化功能。

## 新增功能

### 1. TextProperties - 全面的文本属性结构体

新增 `TextProperties` 结构体，提供类似 minidocx 的属性设置方式：

```cpp
// 创建属性
cdocx::TextProperties props;
props.font = cdocx::TextProperties::Font{"Arial", "SimHei"};
props.fontSize = 36;
props.color = "FF0000";
props.highlight = cdocx::TextProperties::Highlight::Yellow;

// 应用到 Run
run.set_properties(props);
```

#### 新增文本格式化选项：

| 功能 | 描述 | minidocx 对应 |
|------|------|---------------|
| **高亮颜色** | 16种高亮颜色 | ✅ RichTextProperties::Highlight |
| **扩展下划线** | 16种下划线样式（单线、双线、波浪线等） | ✅ RichTextProperties::Underline |
| **删除线样式** | Single/Double/None | ✅ RichTextProperties::StrikeStyle |
| **字符缩放** | 1-600%，100=正常 | ✅ RichTextProperties::scale_ |
| **字符间距** | Expanded/Condensed + 数值 | ✅ RichTextProperties::Spacing |
| **位置调整** | Raised/Lowered + 数值 | ✅ RichTextProperties::Position |
| **字体详细设置** | ascii/eastAsia/hAnsi/cs + hint | ✅ RichTextProperties::Font |

#### 便捷方法：

```cpp
run.set_highlight(TextProperties::Highlight::Yellow);
run.set_underline_style(TextProperties::UnderlineStyle::Wave, "0000FF");
run.set_strike(TextProperties::StrikeStyle::Double);
run.set_scale(150);  // 150%
run.set_spacing(TextProperties::SpacingType::Expanded, 40);
run.set_position(TextProperties::PositionType::Raised, 6);
```

### 2. ParagraphProperties - 全面的段落属性结构体

新增 `ParagraphProperties` 结构体，提供更丰富的段落控制：

```cpp
cdocx::ParagraphProperties props;
props.align = cdocx::ParagraphProperties::Alignment::Centered;
props.outlineLevel = cdocx::ParagraphProperties::OutlineLevel::Level1;

// 缩进
ParagraphProperties::Indentation indent;
indent.special.kind = ParagraphProperties::Indentation::Special::Kind::FirstLine;
indent.special.value = 200;
props.indent = indent;

// 应用到 Paragraph
para.set_properties(props);
```

#### 新增段落格式化选项：

| 功能 | 描述 | minidocx 对应 |
|------|------|---------------|
| **大纲级别** | Level1-9, BodyText（用于目录生成） | ✅ ParagraphProperties::outlineLevel_ |
| **分页控制** | keepNext/keepLines/pageBreakBefore/After | ✅ ParagraphProperties |
| **段落边框** | 上/下/左/右边框独立设置 | ✅ ParagraphProperties::borders_ |
| **改进的缩进** | 字符单位或绝对单位 | ✅ ParagraphProperties::Indentation |
| **改进的间距** | Auto/Lines/Absolute | ✅ ParagraphProperties::Spacing |

#### 便捷方法：

```cpp
para.set_outline_level(ParagraphProperties::OutlineLevel::Level1);
para.set_keep_next(true);
para.set_keep_lines(true);
para.set_page_break_before(true);
para.set_page_break_after(true);
para.set_borders(borders_struct);
```

### 3. TableProperties - 表格属性结构体

```cpp
cdocx::TableProperties props;
props.width.type = TableProperties::WidthType::Percent;
props.width.value = 5000;  // 100%
props.alignment = ParagraphProperties::Alignment::Centered;

table.set_properties(props);
```

---

### 4. SectionProperties - 节属性结构体

```cpp
cdocx::SectionProperties props;
props.pageSize.width = 12240;  // A4
props.pageSize.height = 15840;
props.orientation = SectionProperties::Orientation::Landscape;
props.pageMargins.left = 1440;  // 1 inch
props.columns.count = 2;  // 双栏

// 应用到 section XML 节点
props.applyTo(sectPr_node);
```

## API 兼容性

所有新增功能均为增量添加：
- ✅ 现有 API 保持不变
- ✅ 向后完全兼容
- ✅ 新旧 API 可混合使用

```cpp
// 旧 API 仍然有效
run.set_bold(true);
run.set_font_size(24);
run.set_color("FF0000");

// 新 API 可用
TextProperties props;
props.fontStyle.bold = true;
props.fontSize = 24;
props.color = "FF0000";
props.highlight = TextProperties::Highlight::Yellow;
run.set_properties(props);

// 也可以混合使用
run.set_bold(true);  // 旧 API
run.set_highlight(TextProperties::Highlight::Yellow);  // 新便捷方法
```
