### 12. Caption Generation (标题生成示例)

**位置**: `examples/12_caption_generation/`  
**文件**:
- `main.cpp` - 示例主程序
- `CMakeLists.txt` - CMake 配置
- `data/` - 数据目录

**功能演示**:
1. 统计文档中已有图片数量
2. 获取下一个图片编号
3. 在指定段落后插入图标题

5. 验证标题格式
6. 提取标题文本

**格式规范**:
- 格式: "图 X [描述]"
- 字体: 中文宋体(SimSun)，英文 Times New Roman
- 字号: 10.5pt（五号）
- 对齐: 居中

**编译运行**:
```bash
cd build
cmake --build . --target 12_caption_generation
./examples/12_caption_generation/12_caption_generation
```

## 代码示例

### 书签替换示例代码

```cpp
#include <cdocx.h>
#include <cdocx/bookmark_replacer.h>

// 打开模板
cdocx::Document doc("template.docx");
doc.open();

// 创建替换器
cdocx::BookmarkReplacer replacer(&doc);

// 1. 简单替换（保留格式）
replacer.replace_text("NAME", "张三");

// 2. 批量替换
replacer.replace_text_batch({
    {"DATE", "2024-06-15"},
    {"COMPANY", "北京地质研究院"}
});

// 3. 图片替换
cdocx::ImageSize size(400, 300);
replacer.replace_with_image_advanced("CHART", "chart.png", size,
                                      "分析图", cdocx::ImageAlignment::Center);

// 保存
doc.save("output.docx");
```

### 标题生成示例代码

```cpp
#include <cdocx.h>
#include <cdocx/caption_generator.h>

// 获取下一个编号
int next_num = cdocx::CaptionGenerator::get_next_figure_number(&doc);

// 插入标题
cdocx::CaptionGenerator::insert_figure_caption(&doc, para,
                                                "GPR检测结果", next_num);
```
